DigiTools - digital product store

responsive digital product store built React.js, Tailwind CSS, JavaScript(ES6), and DaisyUI where users can browse products, add to cart, remove items, and complete checkout easily.


Technologies Used
React.js
Tailwind CSS
DaisyUI
JavaScript (ES6+)
React Toastify
JSON Data

Features
Add products to cart with live cart count update
Remove selected products from cart
Proceed to checkout clears all cart items
